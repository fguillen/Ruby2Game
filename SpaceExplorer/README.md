# Space Explorer

Simple game made with [Ruby in Fantasy](https://github.com/fguillen/fantasy).

## Instructions

You are a space explorer. Collect the space rubies and don't get cought by the obstacles.

## How to run

    bundle install
    bundle exec ruby game.rb
